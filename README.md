# Projeto - Aprovação de Empréstimos

Projeto desenvolvido para a disciplina de Inteligência Artificial.

## Objetivo

Criar um modelo de classificação utilizando regressão logística para
prever se um empréstimo será aprovado ou negado com base nas
características do cliente.

## Dataset

O conjunto de dados possui as seguintes variáveis:

-   renda_mensal
-   score_credito
-   dividas_ativas
-   emprestimo_aprovado

## Etapas do Projeto

1.  Carregamento do dataset
2.  Análise exploratória dos dados
3.  Preparação dos dados
4.  Treinamento do modelo de regressão logística
5.  Avaliação do modelo (matriz de confusão, accuracy, precision e
    recall)

## Tecnologias utilizadas

-   Python
-   Pandas
-   Numpy
-   Matplotlib / Seaborn
-   Scikit-learn
